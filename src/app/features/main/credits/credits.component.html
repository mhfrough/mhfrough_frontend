The technology, tools, and people behind this site.

This site is designed, built, and maintained by Mohammad Hamza. It's a full-stack application — an Angular front end talking to a NestJS API over REST and WebSockets — and it stands on the shoulders of a lot of excellent open-source work. Thanks to everyone who builds and maintains the projects below.

Frameworks & Runtime

  • Angular – standalone components, signals, and SSR power the front end.
  • NestJS – the back-end framework running the API.
  • Node.js – the runtime behind it all.
  • TypeScript – end-to-end type safety on both client and server.

Data & Real-time

  • PostgreSQL – the primary database.
  • TypeORM – object-relational mapping and migrations.
  • Socket.IO – the live chat and real-time admin updates.
  • Firebase Cloud Messaging – web push notifications.

Design, Type & Icons

  • Inconsolata – the monospace typeface used throughout the site.
  • Bootstrap Icons – the icon set.
  • Figma and Adobe XD – interface design and prototyping.

Services & Infrastructure

  • Render – application and database hosting.
  • Supabase – object storage for media uploads.
  • Resend – transactional email delivery.
  • Google Gemini – AI-assisted chat replies.

Notable Open-source Libraries

  • RxJS – reactive streams across the stack.
  • Angular CDK – drag-and-drop for the leads pipeline and other primitives.
  • Helmet, Passport & class-validator – security, authentication, and request validation on the API.
  • geoip-lite – approximate, privacy-respecting visitor geolocation.
