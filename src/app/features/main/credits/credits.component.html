Credits

The technology, tools, and people behind this site.

This site is designed, built, and maintained by Mohammad Hamza. It is a full-stack application — an Angular front end talking to a NestJS API over REST and WebSockets — and it stands on the shoulders of a great deal of excellent open-source work. The following acknowledges, with gratitude, the projects, services, and people that make it possible.

Frameworks & Runtime

  • Angular – standalone components, signals, the router, reactive forms, and SSR power the front end.
  • NestJS – the back-end framework running the REST API, WebSocket gateways, and module system.
  • Node.js – the runtime behind it all.
  • TypeScript – end-to-end type safety on both client and server.
  • RxJS – reactive streams across the stack.

Data, Real-time & Messaging

  • PostgreSQL – the primary relational database, accessed via the node-postgres (pg) driver.
  • TypeORM – object-relational mapping and database migrations.
  • Socket.IO – the live chat and real-time admin updates, on both client and server.
  • Firebase Cloud Messaging – web push notifications, via the Firebase web SDK and the Firebase Admin SDK.
  • Keyv & cache-manager – caching across the API.

API Security & Validation

  • Helmet – secure HTTP headers.
  • Passport with JWT and local strategies, and @nestjs/jwt – authentication and token handling.
  • class-validator & class-transformer – request validation and DTO transformation.
  • bcrypt – password hashing.
  • @nestjs/throttler – rate limiting, and cookie-parser – secure cookie handling.

Media, Files & Geolocation

  • sharp – high-performance image processing and WebP conversion for uploads.
  • Multer – multipart file-upload handling.
  • geoip-lite – approximate, privacy-respecting visitor geolocation.
  • idb – an IndexedDB wrapper for offline-first client caching.

Design, Type & Icons

  • Inconsolata, IBM Plex Mono, DM Sans, and Cormorant Garamond – the typefaces used throughout the site, self-hosted via Fontsource.
  • Bootstrap and Bootstrap Icons – layout utilities and the icon set.
  • Angular Material & Angular CDK – UI primitives and the drag-and-drop used in the leads pipeline.
  • Figma and Adobe XD – interface design and prototyping.

Progressive Web App

  • Angular Service Worker and @angular/pwa – offline support, installability, and media caching.

Services & Infrastructure

  • Render – application and database hosting.
  • Supabase – object storage for media uploads.
  • Resend – transactional email delivery.
  • Google Gemini – AI-assisted chat replies.

Tooling & Quality

  • Angular CLI and the NestJS CLI – build, serve, and scaffolding.
  • ESLint and Prettier – linting and consistent formatting.
  • Vitest with jsdom, and Jest – client and server testing.
  • Swagger (OpenAPI) – API documentation.
